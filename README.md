# StudyMap — TCU · CGU · TCDF

App de preparação para concursos públicos de controle externo.

**Acesse:** [studymap no ar](https://seu-usuario.github.io/studymap)

## Abas
- 🌌 Mapa de Constelações — 23 disciplinas, 262 subtópicos
- ⏱ Pomodoro com Cockpit de Estudos (método VR)
- 🃏 Flashcards com repetição ativa
- 📅 Cronograma dia a dia — 126 dias, 4 ciclos
- 💡 Dicas por disciplina com Protocolo I
- 🔄 Ciclos de estudo com rotação semanal
- 🎯 Roda da Aprovação (método VR)
- 📕 Caderno de Erros TDAH-friendly
- 📚 PDF + Claude API — flashcards automáticos

## Uso da aba PDF
1. Acesse [console.anthropic.com](https://console.anthropic.com) e crie uma API key
2. Cole a chave na aba 📚 PDF do app
3. Faça upload de qualquer PDF do Estratégia Concursos
4. Claude gera: resumo, ESF, flashcards e marca subtópicos no Mapa automaticamente
